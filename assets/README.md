# Assets do site Morano

Arquivos usados pelo site:

- `morano-hero.jpg`: imagem principal da hero.
- `morano-logo.png`: logo da Morano no header.
- `morano-interior-01.jpg` e `morano-interior-02.jpg`: vistas do mesmo projeto de interiores, agrupadas como Projeto Essenza.
- `morano-fachada-03.jpg` e `morano-fachada-04.jpg`: vistas da mesma residência, agrupadas como Projeto Prisma.
- `morano-fachada-02.jpg`: projeto residencial separado, usado como Projeto Nexo.
- `morano-interior-03.jpg`: imagem de apoio na seção de serviços.
- `morano-fachada-01.jpg`: imagem de apoio na faixa de citação.

Regra editorial: quando imagens forem da mesma casa ou do mesmo ambiente, elas devem aparecer como variações do mesmo projeto, não como projetos diferentes.
